### v0.0.2   
使用`github.com/gin-gonic/gin`和`github.com/jinzhu/gorm`对项目进行优化重构

* 获取package
    ```
    go get "github.com/gin-gonic/gin"
    go get -u "github.com/jinzhu/gorm"
    ```

* 运行
    ```
    cd src/v0.0.2
    go run main.go
    ```

* 项目结构
  ```
  configs         #配置
  controllers     #控制器
  lib             #库文件
  models          #数据模型
  main.go         #主文件
  ```
